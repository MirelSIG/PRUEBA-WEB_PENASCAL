Descripción
----
Este repositorio contiene una propuesta de aplicación web estática que explora usos de la inteligencia artificial en políticas públicas: modelos de perfiles ciudadanos dinámicos. El objetivo es pasar de actuaciones reactivas a políticas anticipatorias mantenidas bajo supervisión humana y con criterios de equidad, transparencia y protección de datos.

Ideas evaluadas
----
1) Réplica inteligente de ecosistemas (Digital Twins)
- Construcción de modelos digitales que integran datos heterogéneos (sensores, imágenes, sonido, registros administrativos) para simular impactos y evaluar políticas antes de su despliegue.

2) Hologramas humanos para asesoría multilingüe y accesible
- Agentes conversacionales y sistemas Voice‑to‑Voice (ASR + TTS) que guían trámites, explican derechos y eliminan barreras lingüísticas.

3) Perfiles ciudadanos dinámicos para priorización de ayudas
- Perfiles vivos que combinan datos socioeconómicos, formativos y territoriales para detectar necesidades, priorizar apoyos y simular escenarios de política pública.

Entrega y uso
----
El código debe entregarse en un repositorio GitHub. Este proyecto incluye los archivos de la web estática, estilos y contenido, y sirve como demostración técnica y conceptual para el concurso.
